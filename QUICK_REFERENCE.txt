## 🔍 CHECKING CONSOLE (F12)

**Open DevTools:**
- **Windows:** F12
- **Mac:** Cmd + Option + I

**What to look for:**
- Red errors ❌ → Problem
- Yellow warnings ⚠️ → Okay
- No messages ✅ → Good!

**Common errors:**
```
Uncaught TypeError... → JavaScript error
Failed to fetch... → API error (okay, uses fallback)
Cannot find module... → Missing file
CORS error... → API issue
```
